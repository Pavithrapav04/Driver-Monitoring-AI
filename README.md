# 🚗 Driver Monitoring System using AI

## 📌 Project Overview

The Driver Monitoring System is a real-time computer vision-based safety solution designed to detect risky driver behaviors such as drowsiness, yawning, and distraction. The system uses advanced AI techniques to continuously monitor the driver through a webcam and provide instant alerts to prevent potential accidents.

This project integrates multiple technologies including MediaPipe for facial and hand tracking, YOLO for object detection, and audio alerts for real-time feedback. It aims to simulate a smart in-vehicle safety assistant that enhances road safety by identifying unsafe conditions before they lead to critical situations.

---

## ⚙️ Key Features

- 😴 **Drowsiness Detection** – Detects closed eyes and prolonged blinking  
- 😮 **Yawn Detection** – Identifies fatigue through mouth opening patterns  
- 📱 **Distraction Detection** – Detects mobile phone usage using YOLO  
- 🔊 **Real-Time Audio Alerts** – Warns the driver instantly  
- 🎥 **Live Video Processing** – Works with webcam/video input  
- ⚡ **Fast & Efficient** – Lightweight and real-time performance  

---

## 🧠 Technologies Used

- Python  
- OpenCV  
- MediaPipe (Face & Hands Detection)  
- YOLO (Object Detection)  
- NumPy  
- Pygame / pyttsx3 (Audio Alerts)  

---

## 🔄 How It Works

1. Captures live video using webcam  
2. Detects face landmarks using MediaPipe  
3. Calculates eye and mouth ratios  
4. Uses YOLO to detect objects like mobile phones  
5. Identifies unsafe behaviors  
6. Triggers audio alerts when risk is detected
